### 4. 实时市场数据改为5列 ✅

**调整内容：**
- 列数: 4列 → 5列
- 列间距: 5px → 4px
- 数据重新分配为5行×5列布局
- 行情时间跨5列居中显示

**数据分布：**
- 第1行(5列): 最新价、开盘、最高、最低、均价
- 第2行(5列): 涨跌、涨跌幅、昨结算、昨收盘、收盘价
- 第3行(5列): 成交量、成交额、持仓量、昨持仓、结算价
- 第4行(5列): 涨停价、跌停价、合约、最小变动、合约乘数
- 第5行(跨5列): 行情时间

**节省空间：** 约 15px

### 5. 操作建议与交易方向合并 ✅

**调整内容：**
- 将"操作建议"和"交易方向"放到一行并排显示
- 使用flex布局，中间用分隔线
- 删除价格建议区域中重复的交易方向显示

**布局效果：**
```
┌─────────────────────────────────────┐
│       操作建议与方向                │
├──────────────┬──────────────────────┤
│ 操作建议     │    交易方向          │
│   买多       │      做多            │
├──────────────┴──────────────────────┤
│ 信心度: 85%  风险等级: 中           │
└─────────────────────────────────────┘
```

**节省空间：** 约 30px（减少一个独立区域）
